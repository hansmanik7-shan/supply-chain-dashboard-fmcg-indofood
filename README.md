## Project Description
This project presents a Power BI dashboard for analyzing supply chain performance in the FMCG industry using a simulated Indofood case study.

## Tools
- Excel (Data Simulation)
- Power BI (Visualization)
- GitHub (Documentation)

## Analysis Scope
- Demand trend analysis
- Inventory vs demand comparison
- Supplier lead time evaluation
- Stockout rate calculation

## Key Insights
- High-demand products have higher stockout risk.
- Supplier lead time impacts inventory availability.
- Data visualization supports better operational decisions.
